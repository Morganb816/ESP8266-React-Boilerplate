# BoilerPlate 
### Description

This boiler plate is setup for user accounts to be made and logged into. It also takes into consideration user sessions and has built that in. 

---
#### Setup
 - Create a database for your project that matches your package name
 - Run npm install
 - Code!
---
#### Technoligies Used
 - Sequelize
 - React
 - Redux
 - Express
 - PostgreSQL
 - Passport
